# Physics Indirectly

A physics-based Phaser game where you achieve goals indirectly through interactions.

## Requirements Met:
- **Indirect Control**: All challenges involve interacting with the environment to influence the target object.
- **Non-gameplay Scenes**: Includes Intro (Menu), Summary transitions, and Credits.
- **3+ Physics Challenges**:
    1. **Jump**: Use blasts to push the ball over obstacles.
    2. **Acceleration**: Use a magnet to pull the ball through a maze.
    3. **Bounce**: Place temporary bouncers to guide a falling ball.
- **Input**: Functional discrete (clicking) and continuous (holding) inputs.
- **Graphics**: Clean, basic drawn shapes.
- **Tweens**: Animated UI elements and feedback effects.
- **Credits**: Present in-game and listed below.

## Credits & Asset Attribution:
- **Game Engine**: [Phaser 3](https://phaser.io/)
- **Physics Engine**: [Matter.js](https://brm.io/matter-js/)
- **Design & Implementation**: m1stD3V
- **Assets**: All graphics are procedurally generated shapes using Phaser Graphics API.

## How to Run:
1. Install dependencies: `npm install`
2. Start development server: `npm start`
3. Build for production: `npm run build`
